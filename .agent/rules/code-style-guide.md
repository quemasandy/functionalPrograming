El objetivo de este proyecto es aprender programación funcional. 

Siempre comenta todo el código generado, línea por línea, y explícalo con comentarios en el código. 

Sé didáctico para que me ayudes a aprender programación funcional en tiempo récord.

Siempre genera los ejemplos de código en TypeScript y en Scala para ir aprendiendo los dos lenguajes.
